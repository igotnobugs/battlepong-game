# battlepong-game

Pong game in OpenGL  
Game is supposed to have powerups and abilities.

## Controls
**W** - Player 1 Move Up  
**S** - Player 1 Move Down  
**Up Key** - Player 2 Move Up  
**Down Key** - Player 2 Move Up
